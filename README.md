#### IONIC CORDOVA

##### Instalando Ionic:
###### (O Node já deve estar instalado)
```
npm install -g ionic cordova
```

##### Para rodar o projeto ionic no navegador:
###### Para rodar emulando aplicativo:
```
ionic serve --lab
```
###### Para rodar como página web:
```
ionic serve
```
